## Quick Cheat Sheet

| Topic | One-Line Answer |
|---|---|
| What is the project? | Full-stack alumni-student connection platform with talks, media, and reporting |
| Frontend | React 19 + Vite + Tailwind CSS v4 + Redux Toolkit + Framer Motion |
| Backend | Node.js + Express v5 + TypeScript |
| Database | SQLite (better-sqlite3) with WAL mode |
| Auth | JWT in HttpOnly cookie + Bcrypt password hashing (12 rounds) |
| File Storage | Cloudinary CDN for images & videos |
| Automation | node-cron runs hourly to auto-update meet statuses |
| Reports | ExcelJS generates styled .xlsx downloads |
| Architecture | Routes → Controller → Service → DAO (layered pattern) |
